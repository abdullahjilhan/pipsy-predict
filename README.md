# 🟢 Pipsy Signals — Crypto & Forex Signal Platform

A live trading signal web app with institutional-grade multi-indicator analysis.

## 🌐 Live Site
[pipsysignal.netlify.app](https://pipsysignal.netlify.app)

## ⚙️ Tech Stack
React · TypeScript · Vite · Supabase · Recharts · Tailwind CSS · shadcn/ui

## 📊 Features
- Live BUY/SELL signals using RSI, EMA, MACD, Bollinger Bands, Stochastic & ADX
- 4-layer signal system — requires 4/6 indicators to align before firing
- Adjustable parameter settings panel with market presets (Crypto, Forex, Stocks, Commodities)
- Signal Accuracy Dashboard tracking MAE & RMSE over time
- Signal Strength Meter (0-6) with color-coded warnings
